Описание проекта:

Из «Бета-Банка» стали уходить клиенты. Каждый месяц. Немного, но заметно. Банковские маркетологи посчитали: сохранять текущих клиентов дешевле, чем привлекать новых.

Банком предоставлены исторические данные о поведении клиентов и расторжении договоров с банком.

Задача: спрогнозировать, уйдёт клиент из банка в ближайшее время или нет.

Цель: Построить модель с предельно большим значением F1-меры. Довести метрику до 0.59. Дополнительно измерить AUC-ROC, сравнить её значение с F1-мерой.

Результаты: Получена наилучшая модель случайного леса на данных без массштабирования с баллансом классов:
Наилучшая n_estimators: 19
Наилучшая depth: 14
F1-мера наилучшей модели на тестовой выборке с балансом классов: 0.6000000000000001
roc_auc наилучшей модели на тестовой выборке с балансом классов: 0.861261107023819
Accuracy модели: 0.859
